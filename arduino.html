Activite 01 - Premiers pas avec la carte Arduino

Arduino est un minuscule circuit imprimé au potentiel gigantesque. Selon la manière dont vous l’approchez, il peut être utilisé pour émettre un signal en code Morse à l’aide d’une LED ou pour contrôler toute les lumières d’un bâtiment. Ses possibilités dépassent de loin tout ce que vous pourriez imaginer.Dans cette première activité, vous allez prendre en main cette carte et faire clignoter une petite LED.

Activité 04 - Le capteur de température

Dans cet atelier, vous allez découvrir comment utiliser un capteur de température avec Arduino. Le capteur que nous allons utiliser s'appelle le DHT11 et il permet de mesurer à la fois la température et l'humidité de l'air. C'est un capteur fiable et facile à utiliser qui convient à de nombreux projets nécessitant une surveillance de la température et/ou de l'humidité. Vous verrez également comment utiliser une bibliothèque qui simplifie l'écriture du code pour votre projet. L'objectif de cet atelier est de vous familiariser avec le fonctionnement de ce capteur et de savoir comment l'utiliser dans vos projets futurs.

Activité 05 - Le moteur pas à pas

Dans cette activité, vous allez apprendre à utiliser le moteur pas à pas. Il s’agit d’une machine électromagnétique qui converti des impulsions électriques en mouvement précis. L’arbre du moteur bouge par petites incrémentations quand la pulsation électrique est appliquée. La rotation du moteur a un lien direct avec l’impulsion électrique. Il en est de même pour la vitesse.
